# Inside Prada

An immersive exhibition project developed with A-Frame.

## Overview
This project is an immersive exhibition for the Prada brand. Building upon and further developing the brand’s style and identity, it extends the brand’s established atmosphere, material aesthetics and display logic into the virtual space. We interpret the brand through exhibits in various formats, including photographs, videos and products. At the same time, we aim to guide visitors to explore the exhibition in greater depth through interactive elements.

The project is designed as a real-time interactive webpage that allows visitors to move through the exhibition environment in the browser.

## Concept
The exhibition is organised into three connected areas:

- **Zone 1** Through images and text displays, this zone introduces Prada’s history and visual identity.
- **Zone 2** Focusing on materials, this zone features Prada’s nylon bags as its centrepiece, accompanied by videos demonstrating the craftsmanship to aid visitors’ understanding.
- **Zone 3** Serving as an open-plan showroom, this area presents fashion looks and accessories within a more atmospheric setting.

Together, the three areas create a continuous visitor journey through Prada’s archive, material language, and display culture.

## Spatial Structure
The exhibition space is organised as a single continuous environment with connected zones rather than fully separated rooms.

- The outer shell is defined by two long side walls and one back wall.
- Side walls include decorative bands, image displays, and lighting elements.
- Freestanding display walls structure the historical and material sections.
- A video wall acts as an interactive visual element within the exhibition sequence.
- The final showroom presents looks and accessories in an open-plan display arrangement.

## Immersive Experience Value

The immersive value of this project lies in spatial exploration and controlled visual interaction. 

Visitors are not only able to view Prada’s products, but also to navigate a brand environment constructed through a combination of colour, texture and visual storytelling. Interactive video walls and exhibits encourage visitor engagement whilst reinforcing the exhibition’s narrative impact. 

In this way, the project combines exhibition logic with contemporary brand experiences, demonstrating how physical spaces can underpin fashion narratives.

## Target Audience
This project is intended for:

- fashion students
- brand followers
- visitors interested in luxury culture, display logic, and immersive storytelling

## Built With

## Interaction
The project includes basic interactive functions:

- first-person navigation
- mouse-based viewing
- hover interaction on the video wall

## Controls
- **W / A / S / D** to move
- **Mouse** to look around
- Hover on the video wall to activate video playback
- Interact with items by clicking


## Assets / Resources
The project uses:

- 3D models of bags, accessories, fashion looks, display furniture, pillars, lighting fixtures, and decorative elements
- brand-related images and video


All assets are stored in the `asset` folder and referenced through GitHub-hosted links in the project.
